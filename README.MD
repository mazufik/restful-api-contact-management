# Restfull API Contact Management
## User Management Requirement

1. User Data
   - Username
   - Password
   - Name

2. User API
   - Register User
   - Login User
   - Update User
   - Get User
   - Logout User

## Contact Management Requirement

1. Contact Data
   - First Name
   - Last Name
   - Email
   - Phone

2. Contact API
   - Create Contact
   - Update Contact
   - Get Contact
   - Search Contact
   - Remove Contact

3. Contact Address Data
   - Street
   - City
   - Province
   - Country
   - Postal Code

4. Address API
   - Create Address
   - Update Address
   - Get Address
   - List Address
   - Remove Address

## Clone Project

`
git clone https://github.com/mazufik/restful-api-contact-management.git

cd restful-api-contact-management

npm install
`

## Run Project

`
npm run dev
`
